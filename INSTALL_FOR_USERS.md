## Troubleshooting

### Problem: "Installer won't run"
**Solution**:
1. Right-click `installer.ps1`
2. Select "Run with PowerShell"
3. If still doesn't work, call me!

### Problem: "Python/Node.js installation failed"
**Solution**:
1. Download Python manually: [python.org](https://www.python.org/downloads/)
2. Download Node.js manually: [nodejs.org](https://nodejs.org/)
3. Run installer again

### Problem: "Can't connect to Tally"
**Solution**:
1. Open Tally
2. Press F1 (Help)
3. Type "Configuration"
4. Go to: Connectivity ‚Üí Client/Server
5. Set "Enable Server" = Yes
6. Set "Port" = 9000
7. Restart Tally

### Problem: "Desktop shortcut doesn't work"
**Solution**:
1. Go to `C:\K24`
2. Double-click `start-k24.ps1`
3. If that doesn't work, call me!

### Problem: "Forgot password"
**Solution**:
- Call me, I'll reset it for you
- (We'll add password reset later)
